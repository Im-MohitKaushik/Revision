What is the difference between the static and the non-static variables of a class explain it with an example?

ans---  Static Members

*Belongs to class

*Can be accessed using
ClassName.memberName

*Static variable initializes with
default values at the time of class
loading.

*All objects of a class share a
single copy of static variables


========================================================

Non-Static Members

*Belong to Object

*Can be accessed using
objectName.memberName

*Non-static variable initializes with
default values at the time of object
creation

*Each object has one local copy of the
non-static variables.
==========================================================
